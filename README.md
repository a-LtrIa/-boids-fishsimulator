# -boids-fishsimulator
一个基于 Pygame 的简单鱼群模拟程序。
一个基于 Pygame 的简单鱼群模拟程序。  
鱼会表现出对齐、避障、聚集和追逐食物等行为，形成自然的群体动态效果。


## 🌟 功能特点

- **鱼群智能行为**：每条鱼根据周围同伴的位置和方向，自动调整自己的游动方向（基于类 Boids 算法）。
- **食物互动**：点击鼠标左键可在随机位置生成鱼粮，鱼会主动游向最近的食物。
- **边界穿越**：鱼从屏幕一侧离开，会从另一侧重新进入。
- **视觉区分**：鱼有三种蓝色变体，便于观察个体运动。

## 🛠️ 环境要求

- Python 3.7 或更高版本
- 必需依赖库：
  - `pygame`

## 📦 安装与运行

1. 克隆本仓库（或下载 `Fishsimu.py` 文件）：
   ```bash
   git clone https://github.com/your-username/fishsimu.git
   cd fishsimu
   ```
## 安装依赖：
  ```bash
  
  pip install pygame
  运行程序：
  python Fishsimu.py
  ```
🎮 操作说明
鼠标左键点击：在点击位置附近随机生成 5 颗鱼粮。
关闭窗口：退出程序。
